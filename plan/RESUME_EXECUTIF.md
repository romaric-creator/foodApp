## 📞 FAQ

**Q: Est-ce que Claude API est fiable?**
A: Oui, 99.9% uptime SLA, utilisé par des milliers d'apps en production. Anthropic est fondée par des anciens d'OpenAI.

**Q: Que se passe-t-il si Anthropic tombe?**
A: Fallback system qui cache les réponses précédentes. Les utilisateurs voient des recommandations mises en cache. Impact minimal.

**Q: Et la data privacy?**
A: Aucun données personnes n'est sauvegardées chez Anthropic. Juste les prompts de requêtes. Conforme RGPD.

**Q: Peut-on switcher de modèle plus tard?**
A: Oui. Architecture découplée. API abstraction layer permet de changer de provider.

**Q: Quel est le cost réel?**
A: ~€500/mois pour 5000 users avec les 5 modules. Bien moins cher que d'embaucher 1 support person.

**Q: Et si l'IA hallucine?**
A: Nous validons toutes les recommendations vs menu réel. Chat peut halluciner, mais c'est OK (support escalade si besoin). Analytics/Forecast data-backed.

**Q: Timeline peut-elle être plus rapide?**
A: Oui, si on coupe Forecasting/Analytics. Recommendations + Chat = 4 semaines (ajout 1 dev).
