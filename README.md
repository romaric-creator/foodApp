# 🥗 Gourmi IQ - Administration App

## 🧪 Système de Tests Automatisés

### 📦 Backend (Express & Microservices)
Accédez au dossier `backend/` et utilisez les commandes suivantes :
- **Tests d'Intégration** : `npm test` (Lance tous les tests Jest)
- **Benchmark Performance** : `node __tests__/perf/benchmark.js` (Simule la charge réelle)

### 💻 Frontend (React & AI Hub)
Accédez au dossier `frontend/` et utilisez :
- **Tests Unitaires & UI** : `npm test`
